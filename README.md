Web-based Roblox account automation tool with proxy support, email verification, and real-time dashboard.

## Quick Start

**1. Clone the repository**

`git clone https://github.com/Sasorii-sc/Roblox-Automation.git`

`cd Roblox-Automation`

**2. Install dependencies**

`pip install -r requirements.txt`

**3. Run the application**

`python main.py`

**4. Login**

- Username: `Sasorii` / Password: `Sasorii`
- Username: `froxyy` / Password: `froxyy`

## Features

- Account creation with random/custom usernames
- Login with existing accounts (accounts.txt)
- Email verification with temporary email
- Proxy support (Decodo or custom list)
- Auto join groups, like games, follow users
- Avatar customization
- Web dashboard with real-time logs
- Discord webhook reports

## Terminal Commands (in web dashboard)

- `/clear` - Clear console
- `/stop` - Stop current operation
- `/read acc` - Show accounts.txt content
- `/list acc` - Count accounts
- `/help` - Show all commands
